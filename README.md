 # Example Projects
 
 - mycoolapp - A simple get started project
 - 02-dev-tools-demo - Between other things, it provides live reload support to the project
 - 03-actuator-demo - Provides endpoints to get project infos
 - 04-command-line-demo - This is just a normal project which we use command-line to run
 - 05-properties-demo - Using application.properties to change some project properties 
 - 21-hibernate-with-service-demo - DAO Hibernate implementation
 - 22-jpa-cruddemo - DAO JPA implementation
 - 23-spring-data-jpa-cruddemo - DAO Spring DATA implementation
 - 24-spring-data-rest-cruddemo - Spring DATA REST methods usage
 - 30-thymeleafdemo-helloworld - A simple Thymeleaf Hello World project
 - 31-thymeleafdemo-helloworld-css - Thymeleaf Hello World with CSS
 - 32-thymeleafdemo-employees-list - Listing Model data
 - 33-thymeleafdemo-employees-list-css - Listing Model data with bootstrap CSS
 - 34-thymeleafdemo-employees-list-db - Listing data from DB
 - 35-thymeleafdemo-employees-add	- Adding data to DB
 - 36-thymeleafdemo-employees-update - Update data from DB
 - 37-thymeleafdemo-employees-delete - Delete data from DB
